ubuntu dependencies

    sudo apt-get -qq update -qy

clone, virtual environment, install and test

    git clone git@gitlab.com:noahhsmith/statespace.git statespace
    cd statespace
    python3 -m venv venv
    . venv/bin/activate
    pip install -e .
    pytest
